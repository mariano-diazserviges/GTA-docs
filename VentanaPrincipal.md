La ventana principal está organizada con una **cinta de opciones** en la parte superior, al estilo Microsoft Office. Cada pestaña agrupa las funciones de un área concreta.

<img src="./Media/VentanaPrincipal.png" alt="Ventana principal">

---

Las pestañas disponibles son:

<div class="table-pestanas" markdown="1">

| Pestaña | Qué puedes hacer aquí |
| --- | --- |
| <img src="./Media/LogoGTA.png" alt="HorasParte" height="15"> **Inicio** | Titulares (empleados), partes de horas, tablas comunes y herramientas de Excel |
| <img src="./Media/LogoGTA.png" alt="HorasParte" height="15"> **Elementos** | Inventario de equipos e infraestructura, licencias Microsoft 365, lotes de facturación |
| <img src="./Media/LogoGTA.png" alt="HorasParte" height="15"> **Financiero** | Seguros, escrituras, información financiera, informes Navision y gestión Taxor |
| <img src="./Media/LogoGTA.png" alt="HorasParte" height="15"> **Inmobiliario** | Fincas catastrales y registrales, entidades inmobiliarias, contratos y escrituras |
| <img src="./Media/LogoGTA.png" alt="HorasParte" height="15"> **Referencias** | Proyectos, tareas, horas imputadas, certificaciones de obra y comparativos de compras |
| <img src="./Media/LogoGTA.png" alt="HorasParte" height="15"> **Microsoft 365** | Grupos, Planner, tareas To-Do y usuarios de la organización |
| <img src="./Media/LogoGTA.png" alt="HorasParte" height="15"> **Tablas** | Maestros de la aplicación: terceros, provincias, municipios, series, tipos, etc. |
| <img src="./Media/LogoGTA.png" alt="HorasParte" height="15"> **Configuración** | Usuarios, roles y permisos, ajustes de la aplicación y registros de auditoría |

</div>

> La cinta se adapta automáticamente a los permisos de cada usuario: las pestañas y botones a los que no tienes acceso no aparecen.
